| 项目 | 是否必需 | 说明 |

## 2. 下载与安装

### 第一步：下载安装包

前往项目的 [Releases 页面](https://github.com/DrEAmSs59/CS2-insight-agent/releases)，找到最新版本，点击下载 **`CS2-Insight-Agent-Setup-x.x.x.exe`**。

### 第二步：运行安装包

双击下载好的 `.exe` 安装包，按照安装向导的提示完成安装。

> [!IMPORTANT]
> **建议安装路径不含中文字符。** 例如：
> - ✅ `D:\CS2-Insight-Agent\`
> - ✅ `C:\Program Files\CS2-Insight-Agent\`
> - ❌ `D:\游戏工具\CS2洞察\` ← 中文路径可能导致问题

> [!NOTE]
> 部分电脑在首次运行安装包时，Windows 可能弹出「Windows 已保护你的电脑」的 SmartScreen 提示。这是因为程序尚未积累大量用户数量的信任记录，**并非病毒**。点击「更多信息」→「仍要运行」即可继续安装。

### 第三步：完成安装

安装完成后，桌面和开始菜单会出现 **CS2 Insight Agent** 的快捷方式，安装完毕。

## 3. 首次启动与版本更新

### 启动程序

双击桌面的 **CS2 Insight Agent** 快捷方式，程序会直接以桌面窗口的形式打开，**无需打开浏览器，也无需手动启动后端服务**。

程序启动后你将看到一个**深色电竞风格**的桌面界面，左侧是导航栏，右侧是主工作区。

### 在线更新

程序内置**自动更新**功能。每次启动时程序会在后台检测是否有新版本：

- 如果有新版本，界面右上角会出现更新提示
- 点击提示后程序会在后台自动下载更新包
- 下载完成后点击「立即安装」，程序会自动完成更新并重启

你的配置信息和 Demo 数据库**不会**因为更新而丢失。

> [!TIP]
> 也可以随时前往 [Releases 页面](https://github.com/DrEAmSs59/CS2-insight-agent/releases) 手动下载新版安装包覆盖安装，同样不影响已有配置和数据。

## 4. 配置 OBS（录制功能必需）

> 如果你只想**解析 Demo 看高光片段列表**，可以跳过这一节。
>
> 如果你想让程序**自动录制高光视频**，则必须配置 OBS。

### 第一步：安装 OBS Studio

如果你还没有安装 OBS：

1. 打开 [OBS 官方网站](https://obsproject.com/download) 或在 Steam 中搜索下载 OBS Studio
2. 点击 **Windows** 下载按钮
3. 安装时一路默认「下一步」即可

### 第二步：开启 OBS WebSocket 服务

CS2 Insight 通过 OBS 的 WebSocket 接口来远程控制 OBS 的录制功能。你需要在 OBS 里开启它：

1. 打开 **OBS Studio**
2. 点击顶部菜单栏 **「工具」→「WebSocket 服务器设置」**

   > 如果你找不到这个选项，说明你的 OBS 版本太旧。请到官网下载最新版 OBS（28.0 以上自带 WebSocket）。

3. 在弹出的窗口中：
   - ✅ **勾选** 「启用 WebSocket 服务器」
   - **服务器端口**：默认 `4455`（不用改）
   - **密码**：点「生成密码」自动生成一个，或自己设一个；**记住这个密码！** 后面要填到 CS2 Insight 里
4. 点 **「确定」** 保存

> [!TIP]
> 如果你嫌麻烦，也可以取消勾选「启用认证」来去掉密码，这样 CS2 Insight 里密码留空就行。但出于安全考虑不推荐这么做。

### 第三步：OBS 场景与游戏捕获

程序会**自动**在 OBS 中创建名为 **`CS2 Insight Recording`** 的场景（如果不存在的话），你无需手动新建场景。

如果你想手动确认或调整 OBS 的游戏捕获来源：

1. 在 OBS 主界面底部的「来源」区域，点击「+」号
2. 选择「**游戏捕获**」（「显示器采集」也可以，但可能录制过程中会被其他画面影响）
3. 取一个名字（比如"CS2"），点确定
4. 在弹出的属性窗口中：
   - **模式**：选择「捕获特定窗口」
   - **窗口**：选择 `[cs2.exe]: Counter-Strike 2`（如果 CS2 没在运行就先打开一次 CS2）
5. 点 **「确定」**

> [!IMPORTANT]
> **请确保 OBS 在整个录制过程中保持打开状态。** CS2 Insight 的自动录制功能会通过 WebSocket 远程控制 OBS 的「开始录制」和「停止录制」。

### 第四步：在 CS2 Insight 中填写 OBS 连接信息

在 CS2 Insight 的设置页面找到 **「OBS 连接」** 区域，填写：

- **地址 (Host)**：`localhost`（本机不用改）
- **端口 (Port)**：`4455`（和 OBS 里一致）
- **密码 (Password)**：填写你刚才在 OBS 里设的那个密码

点击「**测试连接**」按钮。

如果显示 ✅ 连接成功，恭喜你，OBS 配置完成！

如果连接失败，请检查：
- OBS 是否已打开且 WebSocket 已启用
- 端口号和密码是否填对了
- Windows 防火墙是否拦截了端口（一般本机通信不会）

## 5. 配置 AI 模式（可选）

AI 模式可以让大模型为你的每个高光/下饭片段生成**评分 (0-100分)** 和**锐评文案**。

不开启 AI 模式也能正常使用所有功能，只是看不到 AI 点评。

### 支持的 AI 服务商

| 服务商 | 推荐模型 | 价格 |
| --- | --- | --- |
| **DeepSeek**（推荐） | `deepseek-chat` | 极便宜，几毛钱一场 |
| 通义 Qwen | `qwen-plus` | 便宜 |
| 智谱 GLM | `glm-4-flash` | 有免费额度 |
| OpenAI | `gpt-4o-mini` | 较贵 |
| 其他 OpenAI 兼容 API | 各厂商自选 | 视厂商而定 |
| Ollama / LM Studio | 本地模型 | 免费（需要本地算力） |

### 获取 DeepSeek API Key（推荐）

1. 打开 [DeepSeek 开放平台](https://platform.deepseek.com/)
2. 注册/登录账号
3. 进入控制台 → **API Keys** → **创建新的 API Key**
4. 复制生成的 Key（以 `sk-` 开头的一串字符）
5. 充值几块钱就够用很久了

### 在 CS2 Insight 中配置 AI

在设置页面的 LLM 配置区域填写：

- **服务商**：选择 `DeepSeek`
- **模型**：`deepseek-chat`
- **API Key**：粘贴你刚才复制的 Key

配置会自动保存，解析时勾选「AI 锐评」即可启用。

## 6. 核心玩法：解析 Demo

### 什么是 Demo 文件？

Demo 就是 CS2 的比赛录像文件（`.dem` 格式）。你可以从以下途径获取：

- **Steam 个人比赛历史**：CS2 主菜单 → 你的比赛 → 点击某场比赛 → 最下方的「下载」按钮
- **5E 客户端**：5E → 比赛记录 → 选择某场比赛 → 下载 Demo
- **完美世界平台**：完美世界竞技平台 → 比赛记录 → 下载录像
- **HLTV / 比赛网站**：如果你要分析职业比赛

### 方式一：拖拽上传

1. 将 `.dem` 文件**直接拖到**程序的上传区域
2. 等待文件上传与预解析（会显示地图和玩家列表）
3. 在玩家列表中**选择你要分析的玩家**（通常选你自己）
4. 点击**「开始解析」**
5. 稍等片刻（通常 10-30 秒），结果会以**卡片列表**的形式展示

### 方式二：Demo 库（推荐长期使用）

如果你想持续管理大量 Demo，可以使用 **Demo 库**功能：

1. 在设置或侧栏的**「Demo 监听目录」**中，添加你存放 Demo 文件的文件夹路径
2. 程序会自动扫描该目录下的所有 `.dem` 文件并加入库中
3. 在主界面的 Demo 库列表中选择想分析的 Demo
4. 选择目标玩家后点击解析

### 解析结果怎么看

每个被标记的片段会以卡片形式展示：

- **🔥 高光 (Highlight)**：该回合你杀了 3 个人或更多，或触发了颗秒/Clutch/刀杀等特殊情境
- **💀 下饭 (Fail)**：被电击枪击杀 / 被沙鹰爆头 / 被队友击杀等名场面
- **🪦 梗死亡 (Meme Death)**：你打出了 211 高材生、o 系列等「研发」级别发挥
- **🎬 合集 (Compilation)**：亲儿子喂饭、本命苦主、全部击杀/死亡串烧等跨回合合集

如果开启了 AI 模式，还会显示：
- **AI 评分**：0-100 分
- **AI 锐评**：一段犀利又幽默的点评文案

## 7. 一键自动录制

这是 CS2 Insight 最硬核的功能——全自动控制 CS2 回放 + OBS 录制。

### 前提条件

- ✅ OBS 已配置并连接成功（参见 [第 4 节](#4-配置-obs录制功能必需)）
- ✅ CS2 已安装，且程序设置中的 **CS2 路径**正确指向 `cs2.exe`
  - 程序会自动探测 Steam 安装的 CS2 路径
  - 如果没有自动检测到，需要手动填写，例如：
    `D:\steam\steamapps\common\Counter-Strike Global Offensive\game\bin\win64\cs2.exe`
- ✅ OBS 保持打开状态

### 操作步骤

1. 先完成 Demo 解析，得到高光/下饭片段列表
2. **勾选**你想录制的片段（或全选），加入录制队列
3. 在录制队列页面点击**「开始录制」**按钮
4. 程序会自动完成以下所有操作：
   - 🎮 自动启动 CS2，并加载 Demo 录像
   - ⏩ 自动跳转到每个片段对应的游戏时间点
   - 🎥 自动控制 OBS 开始录制
   - ⏱️ 等待片段播放完毕（jump-cut 自动拼接多杀片段）
   - ⏹️ 自动控制 OBS 停止录制
   - 🔁 自动跳到下一个片段，重复上述过程
   - 🚪 全部录完后自动关闭 CS2

> [!WARNING]
> **录制过程中请勿操作电脑！** 程序需要向 CS2 窗口发送键盘指令来控制 Demo 回放。录制期间切换窗口或打字可能导致命令丢失，严重时可能覆盖你的游戏配置。
>
> 建议泡杯咖啡 ☕ 等几分钟，让程序自动完成一切。

### 录制前观战预热（可选）

在开始录制前，程序会弹出**「观战预热」**选项弹窗，你可以勾选需要的观战 cvar：

- `cl_draw_only_deathnotices`：只显示死亡通知，隐藏其他 HUD
- `hud_showtargetid 0`：隐藏目标 ID 显示
- `tv_nochat 1`：隐藏聊天
- 隐藏投掷物轨迹
- 自定义 FOV 等

这些设置会在每个 demo 会话开始时统一注入，录制结束后自动恢复。

### 计划预览

在正式开始录制之前，可以在录制队列页点击**「预览计划」**查看程序生成的完整录制方案，包括每一段的起止时间、是否启用、以及可能的警告信息，方便你提前调整。

### 录制文件在哪里

录制好的视频文件保存在 **OBS 的默认录制路径**中。你可以在 OBS 中通过**「文件 → 显示录像」**快速打开，或在 OBS 的**「设置 → 输出 → 录像」**中查看和修改保存路径。

成片文件名会按照 `玩家名_地图_回合_击杀数` 的格式自动命名。

## 8. 合辑工作台（可选）

> [!IMPORTANT]
> **合辑导出功能需要先配置 FFmpeg。** 如果你只想录制独立片段而不合成视频，可以跳过 FFmpeg 配置，直接使用录制功能。

### 配置 FFmpeg

**第一步：下载 FFmpeg**

前往 [gyan.dev FFmpeg Builds](https://www.gyan.dev/ffmpeg/builds/)（推荐，国内访问较快），下载 **`ffmpeg-release-essentials.zip`**。

也可以从 [FFmpeg 官网](https://ffmpeg.org/download.html) 选择 Windows 构建包下载。

**第二步：解压**

将压缩包解压到一个**路径不含中文**的目录，例如：

```
D:\ffmpeg\
├── bin\
│   ├── ffmpeg.exe   ← 这个就是你需要的
│   ├── ffplay.exe
│   └── ffprobe.exe
├── doc\
└── ...
```

**第三步：在程序中填写路径**

在 CS2 Insight 的设置页面找到 **「FFmpeg 路径」** 配置项，填入 `ffmpeg.exe` 的完整路径，例如：

```
D:\ffmpeg\bin\ffmpeg.exe
```

填写并保存后，合辑工作台的导出功能即可正常使用。程序会自动探测你的显卡支持的硬件编码器（NVENC / QSV / AMF），没有硬件加速时会自动回退到 CPU 编码（libx264）。

### 使用合辑工作台

录制成功的片段会自动进入**合辑工作台**，你可以在这里：

- **拖拽排序**：自由调整各片段的出场顺序
- **配 BGM**：选择背景音乐
- **选转场主题**：设置片段之间的转场效果
- **导出 MP4**：程序调用 FFmpeg 自动合成并导出最终视频

合辑工作台完全可选，不影响单独使用录制功能。

## 9. Demo 库功能

### 监听目录

在设置中配置**「Demo 监听目录」**后，程序可以实时扫描该目录下的所有 Demo 文件并自动入库。

> [!TIP]
> 可以设置多个监听目录。例如把 Steam 的 CS2 replay 目录和 5E 的下载目录都加进去，所有平台的录像统一管理。

### 关注名单

如果你在设置中配置了**「关注玩家名单」**（填入你和你朋友的游戏昵称），那么当 Demo 库中有新文件进入时，程序会自动识别名单上的玩家，并在库列表中展示他们的 K/D/A 信息，方便你快速筛选想看的录像。

## 10. 常见问题 FAQ

**Q：程序打不开 / 安装后没有快捷方式？**

- 确认安装包已经正常运行到完成，而不是中途关闭了
- 检查 Windows「开始菜单」中是否有 CS2 Insight Agent 的条目
- 尝试重新运行安装包

**Q：启动时弹出 Windows SmartScreen 警告？**

这不是病毒，是因为程序尚未积累足够多的用户信任记录。点击「**更多信息**」→「**仍要运行**」即可正常启动。

**Q：如何更新到新版本？**

程序内置自动更新：启动后若检测到新版本，界面会弹出更新提示，点击下载并安装即可，配置和数据不会丢失。

如需手动更新，也可以直接到 [Releases 页面](https://github.com/DrEAmSs59/CS2-insight-agent/releases) 下载新安装包重新安装，同样不影响已有配置和数据。

**Q：OBS 测试连接失败？**

- 确认 OBS 已经打开并且 WebSocket 服务器已启用（工具 → WebSocket 服务器设置）
- 检查端口号（默认 `4455`）与密码是否匹配
- 确认 OBS 版本 ≥ 28.0（旧版需要单独安装 obs-websocket 插件）
- 尝试在 OBS 中关闭认证（取消勾选），密码留空

**Q：CS2 路径没有自动检测到？**

手动找到 CS2 的安装路径，默认 Steam 安装一般在：

```
C:\Program Files (x86)\Steam\steamapps\common\Counter-Strike Global Offensive\game\bin\win64\cs2.exe
```

如果你把 Steam 游戏库装在了其他盘（比如 D 盘）：

```
D:\steam\steamapps\common\Counter-Strike Global Offensive\game\bin\win64\cs2.exe
```

在设置的「CS2 路径」中填入完整路径即可。

> [!CAUTION]
> **⚠️ 需要特别注意 ⚠️**
>
> **为了保证录制以及还原玩家的原有设置，点击录制开始后请不要手动干预录制过程，鼠标键盘都不要再动！** 若中途有干预录制或在控制台输入，可能会导致玩家本机的游戏设置被覆盖。

**Q：实验性功能 POV HUD 是什么？**

POV HUD 能够让 Demo 的画面看起来更接近实际个人游戏。
在设置页面的「常用参数管理」→「实验性功能」中可勾选 **POV**。开启后，本地 Demo 回放录制时会临时安装项目自带的 `pov.vpk`，并增量修改 CS2 `game/csgo/gameinfo.gi` 的搜索路径；**录制结束后程序会自动恢复**。

- 开启 POV 后，部分与 HUD / 雷达相关的预热选项会被隐藏并由程序强制设定
- 不要用于连接官方匹配服务器
- 若异常退出导致修改未恢复，可在同一页面点击「恢复 POV 修改」（需先关闭 CS2）

**Q：录制时 CS2 没有画面 / OBS 录到黑屏？**

- 确保 OBS 的「游戏捕获」来源设置正确（捕获 `cs2.exe` 的窗口）
- 尝试在 OBS 的「游戏捕获」属性中把模式改为「捕获任何全屏应用程序」
- 确认 CS2 没有被最小化到后台
- 如果使用了多显示器，确保 CS2 和 OBS 捕获的是同一个屏幕

**Q：成片有黑边？**

- 你的 CS2 是 4:3 显示时，有可能发生该情况
- 确保在 OBS 的「游戏捕获」中右键 → 变换 → 拉伸到全屏

**Q：最终效果把控制台录进去了？**

- OBS「设置 → 输出 → 录像质量」不能选「与串流画质相同」，请单独设置录像质量

**Q：录制时 CS2 停留在游戏主菜单？**

目前最新的 CS2 版本仅能播放 2026 年 4 月 21 日（ag2 动作系统更新）以后的 Demo。如果想录制更早比赛的镜头，需要把 CS2 版本回退到 1.41.4.1。

**Q：录制出来的视频是第三人称视角？**

程序会自动切换成第一人称视角来观看目标玩家。如果出现问题，请在解析结果中确认选择的目标玩家名称与 Demo 中的名称完全一致。

**Q：解析很慢怎么办？**

- 第一次解析会稍慢，后续会快很多（Demo 库有缓存）
- 建议不要同时解析多个超大 Demo
- 如果你开启了 AI 模式，AI 点评需要网络请求，视 API 响应速度可能需要额外几秒

**Q：AI 模式报错？**

- 检查 API Key 是否正确（以 `sk-` 开头）
- 检查你的 API 账户是否还有余额
- 如果使用 DeepSeek，确认 Base URL 为 `https://api.deepseek.com`
- 不开启 AI 模式不影响其他所有功能

**Q：我是用 5E / 完美平台打的比赛，Demo 文件在哪？**

- **5E**：5E 客户端 → 比赛记录 → 选择某场比赛 → 下载 Demo
- **完美世界**：完美世界竞技平台 → 比赛记录 → 下载录像
- 下载的 `.dem` 文件可以直接拖入 CS2 Insight 使用

**Q：可以同时录多场 Demo 的片段吗？**

可以。将来自不同 Demo 的片段都加入录制队列，程序会自动按 Demo 分组，依次启动 CS2 并录制各自的片段，全程无需人工干预。
