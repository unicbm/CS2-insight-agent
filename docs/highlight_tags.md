# 片段类型与标签

程序解析 Demo 后，会为每个片段打上情景标签，方便你在列表里一眼看出「这波是什么节目效果」。同一片段可能同时有多个标签；含义相近时只保留更具体的那条（例如已有「颗秒」就不再单独显示「爆头」）。

---

## 片段类型

| 类型 | 是什么 | 什么时候会出现 |
| --- | --- | --- |
| **高光** | 打得好的回合或操作 | 单回合多杀，或单杀刀杀 / 跳杀 / 颗秒；极限拆包、偷包等 |
| **下饭** | 死得离谱或打得抽象 | 死亡时至少命中一种下饭标签；与高光重复时优先保留高光 |
| **合集** | 跨多回合剪成一条 | 见下文「跨回合合集」 |
| **梗局** | 整局 K/D 过于抽象 | 在玩家选择与结果页显示研发标签，可触发 AI 整局毒舌总评（不拆成一条条死亡片段） |

### 局级研发标签（看整局战绩）

| 战绩大致情况 | 标签 |
| --- | --- |
| 2 杀 11 死 | 🎓 211高材生 |
| 0 杀 | 🥚 o{死亡数}、👨‍🔬 首席研发工程师 |
| 1 杀 18 死 | 🗿 i18、👨‍🔬 首席研发工程师 |
| 1 杀（其他） | 👨‍💻 i{死亡数}、👨‍🔬 首席研发工程师 |
| 2 杀（其他） | 💤 z{死亡数}、👨‍🔬 首席研发工程师 |

## 高光标签

### 多杀与节奏

| 标签 | 含义 |
| --- | --- |
| 五杀 (ACE) / 四杀 / 三杀 / 双杀 | 本段内击杀人数 |
| 爆发刷屏 | 短时间内连续完成多杀 |
| 枪枪爆头 | 段内每次击杀都是爆头 |
| ⚔️ 首杀 | 本回合第一个击杀由你完成 |
| 🔪 刀杀 | 含刀杀（含单独成片的刀杀回合） |

### 残局

| 标签 | 含义 |
| --- | --- |
| 🔥 1vN 史诗残局 | 只剩你一人，却打光对方多人 |
| 🔥 2vN 兄弟齐心 | 两人存活面对多人并清场 |
| 🔥 3v5 绝地反击 | 三人打五人并完成多杀 |
| 🐂 1v1 斗牛 | 残局打到 1v1，亲手击杀最后一名敌人赢下回合（仅胜局） |
| 🪖 一人成军 | 队友存活却都不在身边时，孤身被多名敌人（≥3）围攻并完成多杀（≥2，与地图无关，纯相对距离判定） |
| 🔔 极限操作 | 击杀发生在回合尾声或 C4 即将爆炸时 |
| ❤️ 极限锁血战神 | 最后一杀时血量极低 |
| ❤️‍🩹 残血绝地反击 | 多杀开始时血量已经很低 |

### 经济与手枪局

| 标签 | 含义 |
| --- | --- |
| 🔫 手枪局专家 | 第 1 回合或换边后手枪局 |
| 💸 ECO翻盘 | 己方穷局、对方满配，仍打出多杀 |
| 🔫 ECO特种兵 | 己方满配、对方 ECO 局 |
| 👢 光脚干皮鞋 | 己方装备很差、对方很贵，仍有击杀 |

### 比分与赛点

| 标签 | 含义 |
| --- | --- |
| 🛡️ 赛点救世主、命悬一线 | 对方赛点、差一分、本回合有击杀 |
| 📈 绝地追分、拒绝下班 | 对方赛点、分差更大、本回合有击杀 |
| 🗡️ 赛点终结者、一锤定音 | 己方赛点且差一分终结 |
| ⚔️ 加时生死战、大心脏 | 双方 12 分以上且同分 |
| 🔥 顺风局战神、无情碾压 | 大比分领先仍继续收割 |
| ⛰️ 天王山之战 | 双方 10 分以上且同分 |

回合输掉时，可能额外出现「饮恨」类标签，如 😤 1v2 饮恨、💀 1vN 封神未遂、💸 ECO反击、🛡️ 赛点失守、📉 绝地追分未果、⛰️ 天王山饮恨；部分高光会多录一会儿死亡画面，保留结局镜头。

| 标签 | 含义 |

| ✈️ 飞天盲狙、冷神附体 | 空中盲狙 |

| 🔙 背刺 | 刀杀时绕到背后 |
| 🧾 上回合的债 | 本回合击杀的人里，有人上回合杀过你 |
| ⚰️ 补枪 | 击杀刚被队友打残的敌人 |

## 下饭标签

### 常见死法

| 标签 | 含义 |
| --- | --- |
| 💣 惨遭C4洗礼 | 被已安放的 C4 炸死 |
| 电击处刑 | 被电击枪击杀 |
| 沙鹰爆头 | 被沙鹰爆头 |
| 被刀取辱 | 被刀杀 |
| 自杀 | 非摔落/道具的自伤致死 |
| 道具击杀 | 被雷/火等道具打死 |
| 摔死 | 摔落等环境伤害 |
| 痛击队友 | 被队友击杀 |

### 抽象名场面

| 标签 | 含义 |
| --- | --- |
| 切刀就死 | 架了很久的枪切刀/切道具后立刻暴毙 |
| 人肉吸铁石、保镖无用 | 被爆头时队友离你更近敌人，却先死的是你 |
| 人体描边 | 死前对凶手打了好多枪却没打死 |
| 反向锁头 | 喷子狂扫凶手几乎没掉血 |
| NiKo Play | 背身开枪没秒掉，反被一枪带走 |
| 🗿 僵尸步 | 几乎不动仍被爆头 |
| 🐢 散步流 | 移动很快仍被爆头 |
| 🧲 吸铁石 | 主动走向雷/火致死 |
| 🚪 闪送 | 回合刚开始就死 |
| 🪂 空中遇难 | 腾空时被击杀 |

### 肩并肩（单独成片）

| 标签 | 含义 |
| --- | --- |
| 🧍 肩并肩、🙈 视而不见 | 与敌人贴身站了很久却不打 |
| 👫 同框: {敌人} | 贴身的是哪位对手 |
| ⏳ 持续 {N}s | 贴身站了多久 |

## 跨回合合集

| 合集 | 含义 | 大致门槛 |
| --- | --- | --- |
| 🥩 亲儿子喂饭 | 对同一敌人反复击杀 | 单场对该玩家 ≥8 次击杀 |
| ☠️ 本命苦主 | 被同一人反复击杀 | 单场被该玩家 ≥8 次击杀 |
| 🎬 全部击杀 | 本场你的每一次击杀串烧 | 有击杀即生成 |
| 💀 全部死亡 | 本场你的每一次死亡串烧 | 有死亡即生成 |
| 🎬 回合合集 | 从回合开局连续录到死亡或回合结束 | 可多回合勾选后合并入队 |

---

## 回合时间线

解析后还可打开「回合时间线」：按回合查看击杀与死亡，把某一事件或整回合加入录制队列。时间线上的条目会带上对应回合的高光/下饭标签，便于辨认，但本身不算单独一种「片段类型」。

## 标签去重说明

下列情况只显示更具体的一条，例如：

- 有「颗秒」「枪枪爆头」「零距离」「沙鹰爆头」时，不再单独标「爆头」
- 有「飞天盲狙」时，不再单独标「盲狙」
- 有「背刺」时，不再单独标「偷背身」
